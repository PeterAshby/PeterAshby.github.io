Simulating Orbital Dynamics using Numerical Methods

Built an orbital simulation toolkit using Python and the Runge-Kutta 4th order method with adaptive timestep handling. Simulated 2-body and 3-body gravitational systems, including Halley's Comet, a simplified Jupiter-Saturn system, and full barycentric motion around a common center of mass. Focused on numerical stability, performance optimization, and visualization of physical phenomena.

Monte Carlo Methods for Photon Transport and Random Distribution Sampling

Developed and implemented Monte Carlo algorithms in C to simulate two key physical scenarios:

  • Random number sampling from exponential distributions using both rejection and cumulative methods.
  • Photon propagation through an isotropic scattering atmosphere, accounting for absorption and scattering events.

Outputs were visualized in Python using matplotlib to interpret statistical and physical behavior across distributions.
